## 📖 Description

HabAI est une page de destination moderne et interactive pour une application d'intelligence artificielle générative. Ce projet présente une interface utilisateur élégante avec des animations fluides, un design responsive et des fonctionnalités avancées pour démontrer les capacités d'une plateforme IA.
