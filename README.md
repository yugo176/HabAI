## 📖 Description

HabAI est une page de destination moderne et interactive pour une application d'intelligence artificielle générative. Ce projet présente une interface utilisateur élégante avec des animations fluides, un design responsive et des fonctionnalités avancées pour démontrer les capacités d'une plateforme IA.

## ✨ Fonctionnalités Principales

### 🤖 Capacités IA Présentées
- **Chat IA Intelligent** - Interface conversationnelle avancée
- **Génération de Photos** - Création et amélioration d'images
- **Génération de Vidéos** - Production de contenu vidéo
- **Génération de Code** - Assistance au développement
- **Édition de Photos** - Outils d'amélioration d'images

### 🎨 Interface Utilisateur
- **Design Moderne** - Interface épurée avec thème sombre
- **Animations Fluides** - Effets de parallaxe et transitions
- **Responsive Design** - Optimisé pour tous les appareils
- **Typographie Dynamique** - Effet machine à écrire
- **Gradients Colorés** - Design visuel attractif

### 📱 Sections de la Page
- **Hero** - Section d'accueil avec effet typewriter
- **Avantages** - Présentation des fonctionnalités clés
- **Collaboration** - Intégrations avec outils populaires
- **Services** - Détail des services IA
- **Tarification** - Plans Basic, Premium et Entreprise
- **Feuille de Route** - Développements futurs
- **Footer** - Liens et informations de contact

## 🛠️ Technologies Utilisées

### Frontend
- **React 18.3.1** - Bibliothèque UI moderne
- **Vite 7.1.2** - Outil de build ultra-rapide

### Animations & Effets
- **Framer Motion 12.23.12** - Animations React avancées

- **Typewriter Effect 2.22.0** - Animation de texte

### Routing & État
- **React Router DOM 7.8.1** - Navigation côté client

### Développement
- **ESLint** - Linting et qualité du code

- **Autoprefixer** - Compatibilité navigateurs

## 🚀 Installation et Lancement

### Prérequis
- Node.js (version 18 ou supérieure)
- npm ou yarn

### Installation
```bash
# Cloner le repository
git clone <url-du-repository>
cd brainwave-main

# Installer les dépendances
npm install
```

### Développement
```bash
# Lancer le serveur de développement
npm run dev

# L'application sera accessible sur http://localhost:5173
```

### Build de Production
```bash
# Créer le build de production
npm run build

# Prévisualiser le build
npm run preview
```

### Linting
```bash
# Vérifier la qualité du code
npm run lint
```
